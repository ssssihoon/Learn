## 제네릭 함수 / 클래스

제네릭 함수 : 동일한 프로그램 코드에 다양한 데이터 타입을 적용할 수 있게 일반화 시킨 함수

제네릭 클래스 : 동일한 프로그램 코드에 다양한 데이터 타입을 적용할 수 있게 일반화 시킨 함수

- 표준 입출력을 위한 클래스와 객체 변수 등이 선언됨.
